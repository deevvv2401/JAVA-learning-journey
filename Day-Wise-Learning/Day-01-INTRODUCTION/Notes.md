## First Java Program

```java
public class FirstClass {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```


### Important Points
Java Boilerplate

- Every Java program starts with:

1. class
2. main function
3. File Naming Rule
4. Class name and file name should be same.
    Example:
    FirstClass.java

- JDK

Java Development Kit used for Java development.

- JRE

Java Runtime Environment used to run Java programs.

- JVM

Java Virtual Machine converts bytecode into machine-readable code.
